# ia-widgets

Package interne monorepo pour les composants UI réutilisables “Inventiv Agents”.

## Location

`inventiv-ui/ia-widgets`

## Usage (dans `inventiv-frontend`)

```ts
import { IADataTable, AIToggle } from "ia-widgets";
```

## Conventions

- **Préfixe `IA*`** pour tous les exports publics.
- **Pas de nouveaux widgets sans validation** du besoin + style (voir `docs/ui_design_system.md`).
