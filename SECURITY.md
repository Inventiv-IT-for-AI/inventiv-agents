# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please do **NOT** open a public issue.

Instead, please send an email to **security@inventiv-it.com**.

We will validate the issue and fix it as soon as possible.
